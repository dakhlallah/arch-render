# Render Types — what each one needs

The 12 supported render types. Each note says what to capture in the **Final Render Prompt** and
which clause/style to lean on. Always keep the original concept (see output-format rules).

### 1. Photorealistic exterior building render
Three-quarter eye-level or low hero angle. Full materials, real sky, landscaping, scale figures,
natural daylight or golden hour. Default for residential / villa / tower / commercial exteriors.

### 2. Interior render
Wide-angle at standing eye height (24–35mm feel). Natural light from windows + tasteful artificial
light, realistic furniture, materials (floor, walls, ceiling), soft shadows, cozy/airy mood. Name
the room type and key furniture so scale and function stay correct.

### 3. 2D floor plan → 3D render
Turn a flat plan into a **3D dollhouse / cutaway view** (top-down angled, walls extruded, no roof).
Keep EVERY wall, door, window, stair and room position exactly. Add furniture, flooring and realistic
lighting. State "isometric/axonometric dollhouse, faithful to the plan layout".

### 4. Isometric cutaway view
Clean isometric/axonometric angle, building sliced open to reveal interior floors and circulation.
Keep structure, slabs, columns, stairs. Architectural-model feel, soft studio lighting.

### 5. Site analysis diagram
Top-down / aerial diagram. Show plot boundaries, sun path, wind, access/circulation arrows, zoning,
green areas, context buildings. Clean infographic style, labeled, muted palette + accent colors.
(More diagram than photoreal — use illustration/diagram styling, not photorealism.)

### 6. Concept massing render
Simplified volumetric blocks (massing) of the building/urban form, clay or white-model look, clean
studio background, soft AO shadows. For early concept communication. Keep the massing proportions.

### 7. Facade redesign
Keep the building footprint, floor count and window openings; restyle ONLY the facade (materials,
rhythm, color, screens, balconies) per the requested direction. Show before-style geometry, new skin.

### 8. Landscape & urban design
Wider scene: public space, streetscape, plazas, planting, paving, street furniture, people, cars,
trees. Eye-level or aerial. Emphasize human activity, greenery and materials of the ground plane.

### 9. Night render
Blue-hour or full night. Warm glowing interiors through glass, facade/landscape uplighting, street
lights, subtle reflections on wet paving, ambient city glow. Elegant, high-end mood.

### 10. Construction-style technical visual
Section/elevation or exploded-axon with a technical look: visible structure, slabs, columns, layers,
materials called out, neutral lighting. Can be hybrid (photoreal + diagram overlay) if requested.

### 11. Before & after renovation render
Two-state output: keep the existing structure/footprint, show "after" with new materials, finishes,
openings, landscaping. Match camera/angle to the "before" image so the comparison is honest.

### 12. Client presentation image
Polished hero shot, perfect lighting and landscaping, immaculate detailing, aspirational mood,
clean composition with space for text if it's going on a board. Marketing/billboard quality.
